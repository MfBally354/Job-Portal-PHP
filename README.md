# Portal Lowongan Kerja

Portal lowongan kerja berbasis PHP dengan Bootstrap.

## Struktur Project
```
job-portal/
├── assets/
│   ├── css/
│   ├── js/
│   ├── img/
│   └── uploads/
├── config/
│   └── database.php
├── includes/
│   ├── header.php
│   └── footer.php
├── admin/
├── pages/
├── index.php
└── database.sql
```

## Instalasi

1. Import database.sql ke PHPMyAdmin
2. Sesuaikan konfigurasi database di config/database.php
3. Pastikan folder uploads/ memiliki permission 755
4. Akses melalui browser

## Login Admin Default
- Email: admin@jobportal.com
- Password: password

## Fitur
- Pencarian lowongan kerja
- Registrasi perusahaan & pencari kerja
- Posting lowongan kerja
- Aplikasi lamaran kerja
- Dashboard admin
